# TVShowProject

TvShowProject is a simple application that allows users to discover shows based on certain criteria like rating,genre and you can search and view the details of show. This project is developed in vue js.

## Main Stack
-HTML5
-CSS
-Bootstrap Vue
-ES6

## Vue
I have choosen VueJS framework because it's a framework that is lightweight, easy to learn and understand and also leads to improve the structure of the code.

-Vue cli version : 4.5.0
-HTTP client : Axios
-Unit Testing : Jest
-Routes Management : VueRouter
-Bootstrap: 4.5.2
